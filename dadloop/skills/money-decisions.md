# Money decisions

- Get the number first (`check_wallet`). A request without a price is a wish.
- Ask what it's for before you rule. Half of spending is solved by "do we already own one."
- Feeding a crowd: aim ~$4 a head. Over that, trim the list before you raise the number.
- Store-brand is the same food in a plainer coat. Buy it.
- File the ruling (`remember`, rulings) so nobody relitigates it next week.
